# Diabetes Risk Predictor

This project is intended to predict diabetes risk using a Python-based machine learning workflow.

## Project File

- `ml project visual.py` - main Python script for the diabetes risk predictor.

## How to Run

1. Install Python on your system.
2. Open this project folder in a terminal.
3. Run the script:

```bash
python "ml project visual.py"
```
